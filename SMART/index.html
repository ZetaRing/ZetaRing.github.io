SMART: Self-organized Multitask Accelerate Resilient Learning

In Brief

In 2017, Google published a paper, "PathNet" (Chrisantha Fernando et al., 2017). In this model, a well-trained network can be reused for new tasks. It permits parameter reuse without catastrophic forgetting. Using a genetic algorithm, PathNet can discover which part of the network to reuse.

Inspired by PathNet, we go further in transfer learning and artificial general intelligence (AGI). Instead of using a pre-defined network like PathNet, we allow the neural network to "grow up" (evolution) on an environment (dataset) from a "seed" (initial network) under the limitation of "gene pool" (pre-defined framework and components).

We combined PathNet with network synthesis to achieve SMART. SMART can grow forever and be compatible with all the tasks in theory. Furthermore, we investigate whether such growth has a boundary. We use MNIST to prove that network can be adapted to new tasks without extra building blocks, which means the model converges. I also found that SMART is highly corresponding to brain structure and how human learn new things.

Unpublished Manuscript, 2018
