Research Proposal

Brief:
Inspired by LeCun's paper about the "World Model," I want to build an embodied agent without supervision from scratch in an embodied simulator. The agent only relies on RGBD perception and proprioception. The agent will “grow up” like humans and learn everything gradually. The agent will initially learn self-body control, locomotion, and semantic segmentation, and eventually acquire language abilities. By integrating perception, action, and language under a unified framework, I believe we can create a digital version of humans.
